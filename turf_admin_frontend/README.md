# Turf Admin Frontend

Admin panel for managing turf booking system.

## Features

- Admin Login/Authentication
- Dashboard with management modules:
  - Cities Management
  - Venues Management
  - Turfs Management
  - Slots Management
  - Bookings Management
- Modern UI with gradient theme
- Quick stats overview

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

The admin panel will run on `http://localhost:3001`

## Default Admin Credentials

Use the admin credentials configured in your backend system.

## Build for Production

```bash
npm run build
```

## Tech Stack

- React 18
- React Router DOM
- Axios
- Bootstrap 5
- FontAwesome 6
- Vite
